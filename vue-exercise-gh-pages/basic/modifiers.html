處理互動式行為 v-on 指令

  1. 請將 button 改成 a 標籤，並加上 preventDefault()
  2. 使用修飾符取代
  3. 將 input 加上 Enter 事件
  4. 將範例改成使用縮寫表示
  5. 本例子就是指用修飾符簡化原本js的一些功能
    v-on:click.prevent的prevent就是取消空的超連結預設的連結行為
  6. 如果對input上加入v-on:keyup.enter="reverseText"，則當輸入完文字按下enter則會觸發文字反轉功能

對每個DOM要綁定的事件，都是v-on:開頭，而v-on:可以用@來代替，這種代替的方法在vue中叫修飾符

而對每個DOM要綁定v-bind，可以用:來代替，這種代替的方法在vue中叫修飾符
